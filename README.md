# Hashira Placements – Shamir Secret Sharing Assignment (2026)

This project implements recovery of the secret (constant term) using **Lagrange interpolation** from Shamir’s Secret Sharing (SSS) scheme.  
It also detects wrong data shares among the given inputs.

---

## 🛠️ How to Compile & Run

Make sure you have Java (JDK 17 or later) installed.  

```bash
# Compile
javac Main.java

# Run with Test Case 1
type input1.json | java Main

# Run with Test Case 2
type input2.json | java Main
